💊 Unit 6 · Should / Shouldn't & Modals
📐 should / shouldn't — giving advice
should + base verb = good idea ✓ · shouldn't + base verb = bad idea ✗
You should rest. · You shouldn't eat junk food.
📐 can / can't / must / mustn't
can = ability / permission  ·  can't = not able / not allowed
must = strong obligation  ·  mustn't = prohibition (do NOT!)
⚠ All modals + base verb (no -s, no -ing, no to)

💊 Unit 6 · Conditionals Type 0 & Type 1
⚡ Type 0 — General Truth🔮 Type 1 — Real / Possible Future
UseFacts, science, habits — always trueA future situation that is likely
FormulaIf + Pres. Simple, Pres. SimpleIf + Pres. Simple, will + base
ExampleIf you heat water, it boils.If you rest, you will feel better.
⚠ After if, NEVER use will: ✗ If it will rain → ✓ If it rains
When if-clause comes first → comma: If ..., result.

💊 Unit 6 · Conjunctions
ConjunctionMeaningExample
andAdditionI have a cold and a fever.
butContrastI feel sick, but I went to school.
soResultI have a headache, so I will take a pill.
becauseReasonI stayed home because I was sick.
so = result (what happened)  ·  because = reason (why)
I went to the doctor because I had a fever. (reason → why)
I had a fever, so I went to the doctor. (result → what happened)

🎨 Unit 7 · Simple Past
📐 Part 1 — was / were
Subject✓ Affirmative✗ Negative❓ Question
II was at the gallery.I wasn't there.Was I late?
He / She / ItShe was a dancer.He wasn't famous.Was she good?
You / We / TheyThey were great artists.We weren't there.Were they famous?
🕐 Time words: yesterday · last week/month/year · in 1990 · ...ago · when I was young
📐 Part 2 — Regular Verbs → add -ed
paint → painted · watch → watched · perform → performed · visit → visited
dance → danced (drop e)  ·  study → studied (y→ied)  ·  stop → stopped (double consonant)
✗ Negative: didn't + base verb  ·  ❓ Question: Did + subject + base verb?
